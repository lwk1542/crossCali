# oceancolor_ACNIRv2

#### 介绍
基于SeaDAS开源代码的大气校正程序（包括辐射定标），与之前的版本相比，进行了大幅优化，以节约内存开销和批量数据处理时的时间开销。这目前依然不是一个完美的版本，因为其无法支持numba的计算。
